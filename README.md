# How to run

![](/images/img_01.PNG)
![](/images/img_02.PNG)

## 1. Install dependencies

### Backend
```bash
cd server
npm install
```

### Frontend
```bash
cd frontend
npm install
```

## 2. Run the application

### Start the backend
```bash
cd server
npm run dev
```
Runs the backend server at: http://localhost:9999

### Start the frontend
```bash
cd frontend
npm run dev
```
Runs the frontend at: http://localhost:3000
